2.如何表示结构体变量中的每一个成员：
结构体变量名.成员名；
指针变量名->成员名如

struct student
{
    char name[100];
    int age;
    float score;
};

pst->age=88;//在计算机内部会被转化成（*pst）.age；

动态内存分配概述

1.传统数组缺点：
传统数组长度必须事先制定且只能是常整数，不是变量；
传统数组的内存无法手动分配和释放，数组一旦定义，系统为其分配的空间会一直存在直至函数结束；
数组的长度不能在函数运行期间动态扩充或缩小；
传统数组不能跨函数使用；
2.为什么需要动态内存：
动态内存很好地解决了传统数组的缺陷；
3.动态内存和静态内存的区别：
静态内存由系统自动分配和释放是在栈中分配的；
静态内存以压栈方式分配空间，以出栈的方式释放空间；
动态内存由程序员手动分配和释放，是在堆中分配的，故可以跨函数使用；
4.动态内存分配的方法：
通过malloc函数，它是memory allocate的缩写；
int* p=(int*)malloc(4);//分配了4+4=8个字节；p指针变量（静态分配的内存）占4个字节，p所指向的内存（动态分配的内
存）占4个字节；
注：
1.要使用malloc函数必须添加malloc.h这个头文件；
2.malloc函数只有一个形参，并且形参为整型；
3.数字4表示请求系统为本程序分配4个字节；
4.malloc函数只能返回第一个字节的地址，通过强制类型转换将一变成相应字节数
5.动态分配的内存要及时手动释放，否则会造成内存泄漏（内存越用越少）；但同样不能重复释放该内存，否则程序会崩溃；

指针与数组注意事项

在指针指向一个数组元素时，可对指针做以下运算：
1.加一个整数；
2.减一个整数；
3.自加；
4.自减；
5.两个指针相减，如p1-p2(只有p1和p2都指向同一个数组时才有意义)，由p1-p2可知p1和p2之间有多少个内存单元；
注：指针变量不能相加，相乘除，这样做没有任何实际意义

若指针变量已指向数组中的一个元素，则p+1指向该元素的下一个元素；
注：p+1并非将p的值简单加一而是加上一个数组元素所占的字节数；
数组元素类型 所占字节
float 4
long 4
int 4
char 1
通过定义指针变量的类型将1转化成相应字节数;
int a[10];
int* p;
p=&a[0];
若p的值为2000，则p+1的值为2000+1*4=2004；
(p+i)和（a+i）即为a[i]的地址，则它们都指向a[i]这个元素;
