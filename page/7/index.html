switch用法及break用法

switch(i)
{
case 1:
A;
case 2:
B;
default:
C;
case 3:
D;
}
输入1，执行A B C D;
输入2，执行B C D;
输入3，执行D;
输入4，执行C D;
break 用法:
1.终止距离其最近的for循环；
2.终止switch语句；
3.终止if语句（此时if属于循环内部的语句）；

如何看懂程序

分三步走：
第一步：流程控制，也就是程序的执行顺序；
第二步：语句功能，即这个语句种类及其作用；
第三步：试数，这个是看懂程序的关键；

在我们学习过程中可能会遇到一些小算法，我们可以尝试去对它编程，在解决不了的时候，可能会陷入自我怀疑中，这不是正确
的学习方式；
这时候我们应该去看答案，重要的是把答案看懂，之后照着答案去敲程序并进行调试；如果可以，去修改程序，看看输出结果有
什么不同；
如果连答案都看不懂，就把它背下来，不断去想，不断去敲，这是一个自我蜕变的过程，惟其坚持，方可成功！
