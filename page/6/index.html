scanf输入字符串时注意事项

字符串的输出只需要字符数组首地址即可，系统会沿着第一个字符的地址找下去直至遇到\0；系统这么做的前提是数组为连续分
配的内存；
疑问：
char ch[1或2];
scanf(“%s”,ch);//若输入How are you？为什么输出How；
printf(“%s\n”,ch);

字符数组

定义字符数组与定义数值型数据的方法类似，例如char a[10];
字符数组的初始化（这里说的是定义时并初始化而定义以后单个初始化）：
1.完全初始化
如 char a[13]={‘I’,’ ‘,’a’,’m’,’a’,’C’,’h’,’i’,’n’,’e’,’s’,’e’,’.’};
2.不完全初始化
如 char a[3]={‘I’};
未初始化的元素自动定为空字符（\0）；
3.未初始化
如 char a[3];
输出值为垃圾值；

若{}内提供的初值个数大于数组长度则会报错；因此，在定义字符数组时应估计其实际长度保证实际长度不大于数组长度；
字符数组的结束标志：以空字符(\0)为结束标志；以空字符作为结束标志不会产生任何附加操作或增加有效字符，只起一个供辨
别的标志；
printf(“How do you do?\n”);系统会自动在最后一个字符\n后加一个\0；
char a[5]={“china”};//error 此时数组实际长度为6>5;字符串赋值；
char a[5]={‘c’,’h’,’i’,’n’,’a’};//ok 字符赋值；

字符数组的输出：
1.输出字符不包括空字符；
2.若以%s格式符输出时，printf函数的输出控制项为字符数组名；
3.若数组长度大于数组实际长度也只遇到\0结束；
4.若包含一个以上的空字符，则遇到第一个空字符就结束；
5.可用scanf函数输入一个字符串
如 scanf(“%s”,字符数组名)；

数组

数组作用：为解决大量同类型数据的存储和使用；模拟现实世界；
数组分类:一维数组、二维数组、三维数组；
一维数组定义:为n个变量连续分配存储空间；所有变量数据类型相同；所有变量所占字节相同；
一维数组初始化:
定义时整体赋值，否则需单个赋值；
1.完全初始化
int a[3]={1,2,3};
输出1，2，3；
2.部分初始化:
int a[3]={1,2};
输出1，2 ，0；
未初始化值为0；
3.不初始化：
int a[3];
输出三个垃圾值；
一维数组的输出:
循环输出；
int a[n];//n表示元素的个数；元素为a[0]到a[n-1];
二维和三维数组大体同一维类似；
事实上，不存在多维数组，因为内存是线性一维的；
n维数组可以看作是每个元素是n-1维数组的一维数组；
例如:int a[3][4]可以看作为含有3个元素一维数组，只不过每个元素是含有4个元素的一维数组；
int a[3][4][5]可以看作是含有3个元素的一维数组，只不过每个元素是4行5列的二维数组；
