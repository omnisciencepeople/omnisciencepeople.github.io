指针概述

指针和指针变量：
指针就是地址；地址是内存单元的编号，从0开始的非负整数，范围为04G-1；本质上为一个操作受限的非负整数；
指针变量就是存放内存单元编号的变量即存放地址的变量；
指针和指针变量是两个完全不同的概念，但是我们叙述时通常将指针变量简称为指针，实际上他们含义并不相同；
CPU和内存条之间有3根线：
地址总线：确定内存单元；
控制总线：控制传输方向；
数据总线：传输数据；
若有32根地址总线，则有2的32次方个内存单元（一根地址总线对应两种状态），则有2的32次方个字节，则内存为4G(04G-1);
指针分类：
1.基本类型指针
2.指针和数组
3.指针和函数
4.指针和结构体

用函数判断1到某个数字之间的所有素数个数并输出

#include<stdio.h>
#include<stdbool.h>
bool Isprime(int val);
void traverse(int val);
int main()
{
int val;
char ch;
do
{
printf(“请输入你想要输入的数字:”);
scanf(“%d”,&val);
printf(“输出素数为:”);
traverse(val);
printf(“请问你是否还要继续（Y/N）:”);
scanf(“ %c”,&ch);
}while(ch==’Y’||ch==’y’);
return 0;
}
bool Isprime(int val)
{
int i;
for(i=2;i<=val;i++)
if(val%i==0)
break;
if(val==i)
return true;
else
return false;
}
void traverse(int val)
{
int cnt=0;
for(int i=1;i<=val;i++)
if(Isprime(i))
{
printf(“%d\n”,i);
cnt++;
}
printf(“素数的个数为：%d\n”,cnt);
}

函数变量之全局变量

#include<stdio.h>
int A,B,C;//定义了三个全局变量;必须定义在函数前;
int max();//函数声明
int main()
{
printf(“请输入三个整数(中间以空格隔开):”);
scanf(“%d %d %d”,&A,&B,&C);
printf(“max is %d\n”,max());//A,B,C为全局变量所以在调用max函数时用不到传参，max可以直接使用A,B,C;
return 0;
}
int max()
{
int m;
m=A>B?A:C;
if(C>m)
m=C;
return m;
}

变量的存储方式和生存期

变量的存储方式有两种：静态存储方式和动态存储方式；
静态存储方式：指程序在运行期间由系统分配固定的存储空间的方式；
动态存储方式：指程序在运行期间根据需要进行动态的分配存储空间的方式；
内存中供用户使用的存储空间分为三部分：
1.程序区
2.静态存储区
3.动态存储区
全局变量全部存放在静态存储区，程序执行完毕就释放，在程序执行过程中，它们占据着固定的存储单元而不是动态地进行分配和释放；
在动态存储区中存放以下数据：
1.函数形参
2.自动变量
3.函数调用时的现场保护和返回地址等
对以上数据在函数调用时分配存储空间，函数结束时就释放；在函数执行过程中，这种分配和释放是动态的；如果在一个程序中两次调用同一函数（此函数中定义了局部变量）在两次调用时分配给这些局部变量的存储空间的地址可能是不同的；
局部变量的存储类别：
1.自动变量（auto变量）
函数的形参和函数的局部变量，生命周期和函数相同，函数调用结束后就自动释放这些存储空间；
auto 可省略；
2.静态局部变量
用关键词 “static”指定该变量为静态局部变量
函数结束后该变量所占用的存储的单元不会释放从而该局部变量的值得以保存；
静态局部变量缺点：占内存（长期占用不释放，而不能像动态内存那样一个存储单元可以先后为多个变量使用，节约内存）；
降低了程序的可读性（当调用次数过多时往往弄不清静态局部变量的当前值）；
3.寄存器变量
存储在CPU的寄存器中；
为提高执行效率；
对寄存器的存取速度远高于对内存的存取速度；
全局变量的存储类别
1.在一个文件内扩展外部变量的作用域extern
2.将外部变量的作用域扩展至其他文件extern
3.将外部变量的作用域限制在本文件中static
static声明一个变量的作用：
1.对局部变量进行声明，把它分配在静态存储区，该变量在整个程序执行不释放，其所分配的存储空间始终存在直至程序结束，函数内有效；
2.对全局变量进行声明，则该变量作用域只限于本文件夹不能被其他文件引用；

函数递归之流程解析

#include<stdio.h>
int f(int);//函数声明可不加形参，但必须末尾有引号；
int main()
{
int i=f(3);
printf(“i=%d\n”,i);
return 0;
}
int f(int i)
{
if(i==1)
printf(“哈哈\n”);
else
i= f(i-1);
return i;
}
输出：哈哈
i=1
函数调用流程：
f(3)->i=f(2)->i=f(1)->f(1)输出 哈哈;
return 1->f(1)完成->i=1->return 1->f(2)完成->i=1->return 1->f(3)完成;
跳出f函数 输出1；

函数递归之阶乘

#include<stdio.h>
int f(int i)
{
if(i==1)
return 1;
else
return i*f(i-1);
}

{
int i;
char ch;
do
{
printf(“请输入你想求的阶乘数：”);
scanf(“%d”,&i);
printf(“%d的阶乘为%d\n”,i,f(i));
printf(“你是否还要继续（Y/N）:”);
scanf(“ %c”,&ch);
}while(ch==’Y’);

}

函数递归之汉诺塔

#include<stdio.h>
void hannuota(int i,char A ,char B,char C)
{
if(i==1)
printf(“将编号为%d的盘子从%c移到%c\n”,i,A,C);
else
{
hannuota(i-1,A,C,B);//将i-1个盘从A借助C移到B；
printf(“将编号为%d的盘子从%c移到%c\n”,i,A,C);//将编号为i的盘从A移到C;
hannuota(i-1,B,A,C);//将i-1个盘从B借助A移到C;
}
}
int main()
{
int i;
printf(“请为盘子的个数赋值：”);
scanf(“%d”,&i);
hannuota(i,’A’,’B’,’C’);
return 0;
}
