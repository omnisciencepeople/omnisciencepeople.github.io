数据结构之线性结构

什么是线性结构？
用一根线能将所有点串在一起的结构；

线性结构分类
连续存储（数组）
优点：存储速度快、查找元素效率高；
缺点：需连续的很大的内存、插入删除效率低；
离散存储（链表）
优点：空间无限制，插入删除效率高；
缺点：查找元素效率低、存储速度慢；

线性结构应用
1.栈
一种先进后出的存储结构；
分类：静态栈（顺序栈）、动态栈（链式栈）
算法：压栈、出栈；
应用：
函数调用、中断、表达式求值、缓冲处理、迷宫
2.队列
一种先进先出的存储结构；
分类：链式队列、动态队列；
算法：出队、入队；
应用：
凡是和时间有关的操作都和队列有关；

线性结构的算法
遍历、查找、清空、销毁、删除、求长度、排序、插入

指针、结构体及动态内存分配和释放小复习

指针小复习：
1.指针非法访问：定义一个指针变量并未初始化，此时该指针变量有指向，指向一个我们不知道的内存单元，我们可以访问p即
输出该内存单元的地址，但我们不可访问p所指向的内存单元的内容，因为该内存单元的操作权限未分配给本程序；
2.指针就是地址（内存单元的编号），本质上为一个操作受限的非负整数；
指针变量是存放内存单元地址的变量即存放指针的变量；
3.CPU和内存条之间有三根线：数据总线（传输数据）、控制总线（控制输入输出、控制只读只写或读写）、地址总线（确定内
存单元）；
4.内存释放：并非内存真正释放而是指系统将分配给本程序的内存空间的操作权限收回，里面的数据未流失，只是本程序不可访
问；内存释放后内存中数据依然存在故定义变量需要初始化，否则会得到一个垃圾值；
5.确定一维数组需要知道两个条件：数组首地址和数组长度；
6.无论一个指针变量指向的变量占几个字节，该指针变量只占四个字节；
7.若想要通过其他函数改变主函数中变量的值，实参必须为该变量的地址；
8.int* p=&i;//p保留了i的地址；p指向i；*p就等同于i；
9.数组名为数组首地址，是一个常量；

结构体小复习：
1.定义：将一些基本数据类型组合在一起形成的新的复合数据类型；
2.定义结构体变量时末尾有分号；定义结构体变量时可整体赋值，否则只能单个赋值；
单个赋值时，为字符串赋值需用strcpy函数进行，不能直接赋值；
3.struct student st,* pst;pst=&st;//规定pst->数据成员 就等价于 (*pst).数据成员 就等价于 st.数据成员;
4.结构体变量不能加减乘除，但可相互赋值;
5.调用函数时是直接发送变量还是发送变量地址:
发送变量：耗内存（需发送大量字节）、耗时间;
发送变量地址:省时间、省内存（只需发送4个字节）;

动态内存分配专题：
1.内存分配的两种方式：
从静态存储区分配：
编译器请求系统自动分配和释放，生命周期和程序长存；
从动态存储区分配：
栈（stack）：在函数调用期间分配存储空间，函数结束时释放存储空间；生命周期和函数长存；
堆（heap）：由程序员手动分配和释放，函数结束后内存空间不会被释放故可以跨函数使用；但会存在内存泄露（内存越用越少）和程序崩溃(动态分配的内存被多次释放)等问题；
2.动态内存分配方式：
使用malloc函数
注：使用该函数必须添加malloc.h该头文件；
该函数只能有一个形参；
该函数只会返回所分配内存空间的第一个字节的地址；
3.举例：

下列程序中p指向一个合法单元的是（C）
A.main()
{
int* p;
f(p);
}
int f(int* p)
{
int s;
p=&s;
}
B.main()
{
int* p;
f(&p);
}
int f(int** p)
{
int s;
* p=&s;
}
C.main()
{
int* p;
f(&p);
}
int f(int** p)
{
* p=(int*)malloc(4);
}

数据结构概述

数据结构定义：
我们如何把大量而复杂的问题以特定的数据类型和特定的存储结构将它保存到存储器，及在此基础上实现某个功能（如查找、排序、删除）而执行相应的操作；
数据结构关注的是个体及个体与个体之间关系的保存；

算法：
通俗上，为解决问题的方法和步骤即对存储数据的操作从而解决问题；
在狭义上，算法依附存储结构；
在广义上，无论数据以何种方式存储，对数据的操作是一样的；
算法的评判标准：
1.时间复杂度
程序执行的次数并非时间；
2.空间复杂度
算法执行过程中大概所占用的最大内存；
3.难易程度
4.健壮性
5.正确性

程序=数据存储（数据结构）+对数据的操作（算法）+可被其执行的语言；
数据结构的地位：
1.软件中最核心的课程；
2.软件开发中最基础最重要的理论基础；

NULL含义

NULL本质也为0，但这个0不代表数字0，而是表示内存单元的编号0；
我们计算机规定，以0为编号的存储单元的内容不可读、不可写；
以0为编号的存储单元里面存放了非常重要的数据；
