函数递归之流程解析

#include<stdio.h>
int f(int);//函数声明可不加形参，但必须末尾有引号；
int main()
{
int i=f(3);
printf(“i=%d\n”,i);
return 0;
}
int f(int i)
{
if(i==1)
printf(“哈哈\n”);
else
i= f(i-1);
return i;
}
输出：哈哈
i=1
函数调用流程：
f(3)->i=f(2)->i=f(1)->f(1)输出 哈哈；
return 1->f(1)完成->i=1->return 1->f(2)完成->i=1->return 1->f(3)完成
跳出f函数 输出1；

函数递归概述

定义：函数直接或间接调用自己；
函数为什么可以自己调用自己？
前提：函数的调用是通过栈实现的；
A.一个函数在运行期间去调用其它函数，在运行被调函数前需做三件事：
1.将所有实参、返回地址（下一条指令的地址）等信息传给被调函数保存；
2.为被调函数的局部变量分配内存（包括形参）；
3.将控制转移到被调函数的入口从而执行被调函数；
B.从被调函数转移到主调函数前系统也需做三件事：
1.保存被调函数返回的结果；
2.释放被调函数所占用的内存空间（系统收回被调用函数占用的内存空间的使用权限）；
3.依照被调函数所保存的返回地址，将控制转移到主调函数（从被调函数跳出）；
C.当多个函数相互调用时，按照“后调用先返回”的原则，函数间的信息传递和控制转移必须借助栈来实现。即整个程序运行所需的数据空间安排在一个栈中。每调用一个函数时，就在栈顶分配一个存储区进行压栈操作，每当一个函数退出时，就释放它的存储区即出栈。当前运行的函数永远在栈顶。
D.在计算机看来，函数A调用函数B 和 函数A调用函数A是没有区别的。（执行步骤相同）
注：
不能无限压栈和出栈，否则系统会崩溃；
递归必须满足两个条件：
1.递归必须有明确的终止条件；
2.该函数所处理的数据规模必须在递减；n->n-1->n-2->n-3->…->1
递归和循环的关系：
所有循环都可转化为递归；所有递归不一定能由循环实现；
递归和循环优缺点：
递归：
优点：易于理解；缺点：速度慢、存储空间大（浪费空间多）；
循环：
优点：速度快、存储空间小（浪费空间少）；缺点：不易理解；
递归应用：
树和森林以递归方式定义；
树和图的很多算法都是以递归实现；
很多数学公式都是以递归方式定义的，如斐波拉契数列；

函数概述

为什么需要函数：
1.函数可避免大量重复性代码；
2.有利于程序模块化；
函数定义1：
逻辑上，能完成特定独立功能的代码单元块；
物理上，能接收数据，能对接收数据进行处理，能将处理结果返回；
函数适用范围：
适用大量同类型问题；
函数功能越单一越好（可移植性越强）；
函数定义2：
函数返回值类型 函数名（函数形参）
{
函数执行体
}
定义2本质：
1.详细描述函数之所以能实现某个特定功能的具体实现方法；
2.return表达式：向主调函数返回函数值；为空则表示终止函数；
3.函数返回值类型称函数类型；如果函数返回值类型和return表达式中数据类型不同的话，以函数名前返回值类型为主；（函数类型决定返回值类型）
函数分类：
有参函数和无参函数；
有返回值函数和无返回值函数；
库函数和用户自定义函数；
普通函数和主函数；
值传递函数和地址传递函数；
函数注意问题：
一个程序有且只有一个主函数；
主函数可调用普通函数；
普通函数不可调用主函数；
普通函数可相互调用；
程序从主函数入主函数出；
函数必须先定义后使用；
函数声明：
1.告诉编译器即将出现的若干个字母代表函数；
2.告诉编译器即将出现的若干个字母代表函数具体情况；
3.其为一个语句必加分号；
4.对库函数的声明是通过#include<库函数所在文件名.h>来实现的；
函数形参和实参：
个数相同；位置对应；类型相互兼容；
函数重要性：
函数为C语言的基本单位，类为Java/C++/C#的基本单位；
常见的系统函数：
sqrt:求平方根；
abs：求整型数据的绝对值；
fabs：求浮点型数据绝对值；
变量分类：
1.按作用域：全局变量和局部变量；
2.按存储方式：静态局部变量，自动变量，寄存器变量；
注：在一个函数内部定义的变量或函数的形参称为局部变量；
在所有函数外部定义的变量称全局变量；
全局变量和局部变量命名冲突问题：
在一个函数内部，若定义一个局部变量，其名与全局变量相同，局部变量会屏蔽掉全局变量；
俗称“强龙不压地头蛇”。
例如：
#include<stdio.h>
int i=99;//定义一个全局变量；
void f(int i)//定义一个局部变量，名字和全局变量相同；
{
printf(“%d\n”,i);
}
int main()
{
f(8);//调用f函数；
return 0;
}
//输出8，而非99；

使用字符串处理函数

#include<stdio.h>
#include<string.h>
int main()
{
char str[]={“hello world”};
puts(str);//字符串输出函数相当于printf函数
gets(str);//字符数组赋值函数相当于scanf函数
puts(str);
char str1[30]={“hello”};
char str2[]={“China”};
puts(strcat(str1,str2));//strcat字符串连接函数，函数调用后得到一个函数值：数组str1的首地址；
字符数组1必须足够大以便容纳连接后的字符串；连接前两个字符串后面都有’\0’,连接时将字符数组1后面的\0取消，只在新字符串最后保留’\0’;
strcpy(str1,str2);//字符数组1必须为数组名，字符数组2可为字符数组名也可为字符串常量；
puts(str1);
strcpy(str1,”hello”);
puts(str1);
strncpy(str1,str2,2);//将字符串2最前面的两个字符复制到字符数组1上取代字符数组1原有的最前面的两个字符；
puts(str1);
strcmp(str1,str2);//字符串比较函数：将两个字符串自左到右逐个字符比较（按ASCII码数值大小比较），直至出现不同的字符为止以第一对不相同字符比较的结果为准；若字符全部相同则认为这两个字符串相等；相同函数值为0；字符数组1大于2函数值为正整数；反之为负整数；
strlen(str1);//测字符串长度的函数；
strlwr(str1);//转化为小写的函数；
strupr(str1);//转化为大写的函数；
return 0;
}

scanf输入字符串时注意事项

#include<stdio.h>
int main()
{
char a[100];
scanf(“%s”,&a);//error:字符数组中字符数组名即为该数组第一个元素的地址或者说该数组的起始地址；
scanf(“%s”,a)
printf(“%s\n”,a);//字符数组以%s输出时，输出控制项为字符数组名；
return 0;
}
#include<stdio.h>
int main()
{
char a[100],b[100],c[100];
scanf(“%s%s%s”,a,b,c);//若输入How are you?输出How are you?；若输入Howareyou?输出无输出;有空格字符分隔作为三个字符串输入，无空格字符，当作一个字符串输入；
两者a[100],b[100],c[100]不同
printf(“%s%s%s\n”,a,b,c);//字符数组以%s输出时，输出控制项为字符数组名；
char ch[100];
scanf(“%s”,ch);//若输入How are you？//输出How；系统将空格字符当作字符串间的分隔符故输出How
printf(“%s\n”,ch);
return 0;
}
//字符串的输出只需要字符数组首地址即可，系统会沿着第一个字符的地址找下去直至遇到\0；系统这么做的前提是数组为连续分配的内存；
疑问：
char ch[1或2];
scanf(“%s”,ch);//若输入How are you？为什么输出How；
printf(“%s\n”,ch);

字符数组

定义字符数组与定义数值型数据的方法类似，例如char a[10];
字符数组的初始化（这里说的是定义时并初始化而定义以后单个初始化）：
1.完全初始化
如 char a[13]={‘I’,’ ‘,’a’,’m’,’a’,’C’,’h’,’i’,’n’,’e’,’s’,’e’,’.’};
2.不完全初始化
如 char a[3]={‘I’};
未初始化的元素自动定为空字符（\0）；
3.未初始化
如 char a[3];
输出值为垃圾值；

若{}内提供的初值个数大于数组长度则会报错；因此，在定义字符数组时应估计其实际长度保证实际长度不大于数组长度；
字符数组的结束标志：以空字符(\0)为结束标志；以空字符作为结束标志不会产生任何附加操作或增加有效字符，只起一个供辨别的标志；
printf(“How do you do?\n”);系统会自动在最后一个字符\n后加一个\0；
char a[5]={“china”};//error 此时数组实际长度为6>5;字符串赋值；
char a[5]={‘c’,’h’,’i’,’n’,’a’};//ok 字符赋值；

字符数组的输出：
1.输出字符不包括空字符；
2.若以%s格式符输出时，printf函数的输出控制项为字符数组名；
3.若数组长度大于数组实际长度也只遇到\0结束；
4.若包含一个以上的空字符，则遇到第一个空字符就结束；
5.可用scanf函数输入一个字符串
如 scanf(“%s”,字符数组名)；
