逗号表达式及自增表达式

#include<stdio.h>
int main()
{
int a;
int i=1;
a=(i++,++i,i+3,i+5);
printf(“%d”,&a);
return 0;
}
/*
输出8
i++先赋值再自增；
++i先自增再赋值；
（A,B,C,D）从左到右依次执行，逗号表达式最终值为最后一项；
*/
