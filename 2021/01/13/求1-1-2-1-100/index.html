求1+1/2+...+1/100

#include<stdio.h>
int main()
{
int i;
float sum=0;
for(i=1;i<=100;i++)
sum=sum+1.0/i;//或者为sum=sum+1/float（i）；
printf(“sum=%f\n”,sum);
return 0;
}
/*
整形数据/整形数据，结果为整形；
除法运算符中若有至少一个浮点型，结果为浮点型；
商的符号由除数和被除数决定
因此对上述sum求值过程中进行强制类型转换；

取余运算符符号只和被除数符号相同；
*/
