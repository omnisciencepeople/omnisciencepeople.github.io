互换两个数字

#include<stdio.h>
int main()
{
int a,b,t;
scanf(“%d %d”,&a,&b);
printf(“赋值后的结果为：a=%d b=%d\n”,a,b);
t=a;
a=b;
b=t;
printf(“互换后的结果为:a=%d b=%d\n”,a,b);
return 0;
}
/*
互换两个值需要中间值的存在，我们要以计算机的思维考虑问题;
