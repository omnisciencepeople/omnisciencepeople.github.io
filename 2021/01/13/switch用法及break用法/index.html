switch用法及break用法

switch(i)
{
case 1:
A;
case 2:
B;
default:
C;
case 3:
D;
}
输入1，执行A B C D;
输入2，执行B C D;
输入3，执行D;
输入4，执行C D;
break 用法:
1.终止距离其最近的for循环；
2.终止switch语句；
3.终止if语句（此时if属于循环内部的语句）；
