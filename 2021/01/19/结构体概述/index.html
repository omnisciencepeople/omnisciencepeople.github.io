结构体概述

为什么需要结构体？
为表示复杂的事物，普通数据类型无法满足要求；
结构体定义1：
把一些基本类型组合在一起形成一个新的复合数据类型；
结构体类型变量的定义：

1.先声明结构体类型再定义该类型的变量

3.省略结构体类型名（不提倡使用）

如何使用结构体变量：
1.赋值和初始化：
定义时初始化可整体赋值如

定义后只能单个赋值如

2.如何表示结构体变量中的每一个成员：
结构体变量名.成员名；
指针变量名->成员名如

struct student
{
    char name[100];
    int age;
    float score;
};
struct student st={"张三",18,100}；
struct student*pst=&st;
pst->age=88;//在计算机内部会被转化成（*pst）.age；
pst->age含义：pst指向结构体中age这个成员；
3.结构体变量的运算：
不能相加减乘除但可以相互赋值；
