java线程演示

1.创建线程的第一种方法：
创建一个Thread继承类并重写Thred类中的run方法，
创建Thread类对象，通过该类从Thread继承过来的start方法调用run方法，
直接调用run方法是不会开辟出新的线程的

2.创建线程的第二种方法：
创建一个实现了Runnzble接口的类并重写该接口中的唯一方法run方法，
创建一个该类的对象，
利用该类对象创建一个Thread对象，
调用Thread对象的start方法（start方法会开辟新的线程并自动调用实现了Runnzble接口的类对象的run方法）
